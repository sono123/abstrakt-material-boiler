Abstrakt Material

Version: 0.0.1

Abstrakt Material is a bootstrap extention that combines aspects of Material design and Abstract design.

This library rides on top of the Bootstrap 4 front-end library.

* Note, this project is mostly just a portfolio piece and not a real project. Feel free to use it, just keep in mind it will not be maintained often.

Introduction

Abstrakt Material is a design style that combines elements from Material Design and Abstract Design. As you can see below, both Material and Abstract styles have their own approach to shadows and dimensions. Abstrakt Material takes aspects from both, resulting in a flat design language that incorporates some degree of shadows and dimension.

Abstract Design

Abstrakt Material

Abstrakt Material is strictly a styling add-on. The grid system and layout is entirely Bootstrap 4. The focus of this library is to keep the amount of code as minimal as possible, meaning most of the styles apply to Forms, Buttons and Cards. Furthermore, some of the classes are designed to coexist with Bootstrap 4 classes, not replace them.

Getting Started

The easiest way to use Abstrakt Material is to use the CDN by including the following line of code in the head of your document. Make sure this library is included after Bootstrap in your application.

Forms

On forms, Abstrakt Material styles are limited to text inputs, textareas, select tags, radio inputs and checkboxes. Buttons are styled as well, however they do not need to be inside of a form. The example below is broken into individual elements further down the page.

Inputs

For text input styles, make sure your input element is wrapped inside of a parent element with the classes .form-group .row .am-form-row. Also, the styling will not work consistently unless the aforementioned elements are located inside of a form.

Code

<form>
  <div class="form-group row am-form-row">
    <label for="email">Email</label>
    <input type="email" class="form-control" id="email" placeholder="Email">
  </div>

Radio Buttons

For radio button styles, add the class .form-check-input to the input element. Add a wrapper around the input element with the classes .form-check .am-form-check. Finally, both aforementioned elements should be placed inside an element with the classes .form-group .row .am-form-row. Also, the styling will not work consistently unless the aforementioned elements are located inside of a form.

Code

<form>
  <div class="form-group row am-form-row">
    <div class="col-sm-12">
      <div class="form-check am-form-check">
        <input type="radio" class="form-check-input" name="optionsRadios" id="radio-one" value="option1" checked>
        <label for="radio-one" class="form-radio-label">Radio 1</label>
      </div>
    </div>
    <div class="col-sm-12">
      <div class="form-check am-form-check">
        <input type="radio" class="form-check-input" name="optionsRadios" id="radio-one" value="option1" checked>
        <label for="radio-one" class="form-radio-label">Radio 1</label>
      </div>
    </div>
  </div>

File Fields

For file field styles, make sure your input element has the classes .form-control-file .am-file-input, and is wrapped inside of a parent element with the classes .form-group .row .am-form-row. Also, the styling will not work consistently unless the aforementioned elements are located inside of a form.

Code

    <label for="exampleInputFile">File Field</label>

Panels

A panel is essentially the same as a card. Panels are flexible and extensible content containers. Although Bootstrap already has cards within its library, Abstrakt Material has its own unique style and format, which can be used as an alternative to Bootstrap cards.

To use panels, start by adding the class .am-panel-wrapper to the Bootstrap grid class. Inside that grid element, add the class .am-panel to activate a panel. You can also specify the background color by adding a class such as .white. To add titles and content to the panel, add the respective classes .am-card-title and .am-card-content.

Code

<div class="row">
  <div class="col-sm-12 col-md-4 am-panel-wrapper">
    <div class="am-panel white">
      <h5 class="am-card-title">Card Title</h5>
      <img src="https://placeimg.com/450/300/animals">
      <p class="am-card-content">Card content goes here.</p>
    </div>
  </div>
<div>

Images

Abstrakt Matarial adds styles to images which do not adversely interact with Bootstrap 4 native image styles. Essentially, the only style change that occurs is the addition of an abstract style of shadow below the image.

To use use Abstrakt Material image styling simply add the class .am-image. Note, although the Bootstrap class .img-thumbnail will work with Abstrakt Material image styles, the look is not optimal.

Code

<div class="col-sm-12 col-md-4">
  <img src="https://placeimg.com/450/300/animals" class="am-image">
<div>

Text Content

Text content in Abstrakt Material uses the default Bootstrap 4 styles. This can easily be overwritten, however consider the sizing and alignments may be adversely affected.
